Option walk_corridor

Uses the data from the laser sensor to walk through a corridor as good as possible, without bumping into the walls

State Machine

State decide_movement

If that state is active,
 >  The basic behavior differential_drive is executed. Parameters:
    differential_drive.speed  =  0 ;
    differential_drive.turning_speed  =  0 ;
 >  The basic behavior wait is executed. Parameters:
    wait.time  =  2 ;
 
The decision tree:
 
 
Pseudo code of the decision tree:
 
/** */
if ( laser_max() < maximum_laser_value)
{
  
goto move_back;
}
else
{
  
/** */
if ( laser_max() == laser_min_n() )
{
  
goto move_forward;
}
else
{
  
/** */
if (
(laser_max() == laser_min_nne() )
|| 
(laser_max() == laser_min_ne() )
|| 
(laser_max() == laser_min_ene() )
)
{
  
goto move_right;
}
else
{
  
/** */
if (
(laser_max() == laser_min_nnw() )
|| 
(laser_max() == laser_min_nw() )
|| 
(laser_max() == laser_min_wnw() )
)
{
  
goto move_left;
}
else
{
  
stay;
}
}
}
}

State move_forward

If that state is active,
 >  The basic behavior differential_drive is executed. Parameters:
    differential_drive.speed  =  forward_speed ;
    differential_drive.turning_speed  =  0 ;
 
The decision tree:
 
 
Pseudo code of the decision tree:
 
/** */
if ( laser_max() == laser_min_n() )
{
  
stay;
}
else
{
  
goto decide_movement;
}

State move_left

If that state is active,
 >  The basic behavior differential_drive is executed. Parameters:
    differential_drive.speed  =  forward_speed ;
    differential_drive.turning_speed  =  turning_speed ;
 
The decision tree:
 
 
Pseudo code of the decision tree:
 
/** go on till threat is over */
if (
(laser_max() == laser_min_nnw() )
|| 
(laser_max() == laser_min_nw() )
|| 
(laser_max() == laser_min_wnw() )
|| 
(laser_max() == laser_min_n() )
)
{
  
goto decide_movement;
}
else
{
  
stay;
}

State move_right

If that state is active,
 >  The basic behavior differential_drive is executed. Parameters:
    differential_drive.speed  =  forward_speed ;
    differential_drive.turning_speed  =  (0 - turning_speed) ;
 
The decision tree:
 
 
Pseudo code of the decision tree:
 
/** */
if (
(laser_max() == laser_min_nnw() )
|| 
(laser_max() == laser_min_nw() )
|| 
(laser_max() == laser_min_wnw() )
|| 
(laser_max() == laser_min_n() )
)
{
  
goto decide_movement;
}
else
{
  
stay;
}

State move_back

If that state is active,
 >  The basic behavior differential_drive is executed. Parameters:
    differential_drive.speed  =  (0 - reverse_speed) ;
    differential_drive.turning_speed  =  0 ;

/** */
if ( laser_max() > maximum_laser_value)
